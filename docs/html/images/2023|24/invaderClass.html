8. Add Animation - Make your Invader Wave in a Threatening Manner!

Give your Invader class another characteristic, an isAlive variable that is boolean and set to true.

boolean isAlive = true;       // <-- Invader status.

Then, update the Invader's draw() method as shown, to change the image depending upon:

  • the value of isAlive,
  • and if this is true, vary the image that is shown based upon frameCount and animSpeed.

You may want to create a local variable to make this more readable - I use and calculate whichImage.

  void draw()
  {
    int whichImage = (frameCount/animSpeed)%2; // <-- uses modulo to return 1 or 0 from value that varies with frameCount

    if (! isAlive) {
      tint(255, 64);
      image(img2, x, y, size, size);
    } else {
      if (whichImage == 1) {
        image(img1, x, y, size, size); // <-- vary image shown depending on value of whichImage
      } else {
        image(img2, x, y, size, size); // <-- vary image shown depending on value of whichImage
      }
    }
  }
