10. Make your Invader Stay on The Screen!

We need the Invader to stay on the screen, so add a method that will change the direction of travel when the Invader reaches the screen edge.

A good way to do this is to add a checkForEdge() method to the Invader class and call this before you draw() the Invader.
The method should change the speed to the negative of the current speed if the Invader is at an edge - this will effectively make it change direction: speed = speed * -1.

Update the sketch to checkForEdge() after the Invader move()s in the main draw() loop.

  void checkForEdge()
  {
    if (condition) // <-- under what condition(s) do we want to change the direction of travel of the Invader?
    {
      speed = speed * -1;
    }
  }
