Some short videos that introduce some of the capabilities of the PImage in Processing.

So watch them through and try out the ideas.

The imageGrid application is available in the Images lecture. It is intended to help you explore the overall concept of the 'image' as a structured set of numbers representing colour defined according to their red, green and blue content.

Be sure to spend time experimenting with it.

The API used to code in Processing is a little old, and both Moodle and the Processing documentation have had something of a re-vamp since the video was recorded, but the concepts and the code are exactly the same, so don't let this put you off.

Animate Invader - 6:03

Animation involves loading image sequences successively.
Here this is achieved with the frameCount variable and % (modulo) operator to bring a space invader to life.

Loading & Using Images

Here we use loadImage() to load an image from the web into a PImage that we can use in our Processing sketch.
We can then display the image on the canvas at a particular position and with a particular size by calling the image() method.

Using & Saving Images

This video involves loading images from your local drive with loadImage() and saving these, and whatever else you have drawn to the canvas, to an image file in your filespace with save().
